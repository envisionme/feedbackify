Feedbackify
================================================================================

This module allows quick implementation of the Feedbackify service code. You need an account with Feedbackify in order to use this module.

How to use
----------------------------------------------------------------------------------------------------------------------------------------------------------------

1. Sign up for a Feedbackify account at http://www.feedbackify.com
2. Grab the form ID from your feedback form (can get this from the URL at http://www.feedbackify.com/app/edit/[form ID] or http://www.feedbackify.com/app/deploy/[form ID])
3. Enable this module
2. Go into admin > settings > feedbackify and paste your copied form ID in the required box
